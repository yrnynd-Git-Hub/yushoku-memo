🌙 夕食メモへようこそ

GitHubのGist（無料）にデータを保存します。スマホや別のPCからも同じデータを見られます。GitHubアカウントがあればすぐ使えます。

1
GitHubでトークンを発行

GitHub にログイン → 右上のアイコン → Settings → 左下の Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic)

名前は何でもOK。Expiration: No expirationを選択し、gist にチェックを入れて Generate。

🔗 トークン発行ページを開く
📱 初回（新規作成）
🔗 2台目以降（既存に接続）
🌙
夕食
夜ごはんを記録

✨ 今日の夕食を提案

最近の記録をもとにバランスよく提案します。
記録が増えるほど精度が上がります。

🔗 Gist ID（連携コード）

2台目のスマホやPCで使いたいときは、このIDをセットアップ画面に入力してください。

—

🔄 データを再読み込み

他の端末で記録した内容を手動で反映したいときに使います。

⚠️ 設定をリセット

このデバイスの設定をクリアして初期設定画面に戻ります。Gistのデータは消えません。
